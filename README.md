# Deep-Learning-based-OpenPose-Estimation
Deep learning based OpenPose estimation.  

## Dataset:-  

1. VGG Pose Dataset.  
2. COCO Keypoints challenge.  

## Reference paper:-  

1. https://arxiv.org/pdf/1611.08050.pdf.  


## Important terms used in the reference paper:-

1. Confidence Maps.  
2. L2 Loss. 
3. Affinity fields.  

## Requirements:-  

1. Caffe:- 1.0.  
2. Python 3.6 and above.  
3. OpenCV 3.4.1 and above.  


## Codes available in:-

1. Python.  
2. C++.
